Description of the project

In this project I have Created a Restaurant themed food website using HTML and CSS, ReactJs .Here are insights of my project:

  • Firstly, Created a placeorder placeholder to order the favourite food.
  • Next directed to the menu page with various varieties of food items.
  • We can search for the item we want so that we can add the product to cart.
  • After adding to cart we can proceed to the payment.
