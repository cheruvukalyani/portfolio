Description of the project

In this project I have Created a movies theme website using HTML,CSS and JavaScript.Here are insights of my project:

  • Firstly, Created a login page for geting started to the webiste.
  • Next after loging into the platfrom we can choose whether to see movies,series,shortfilms or dramas.
  • In movies we can choose genres like Action,comdey,romantic,thriller and horror.
  • In series also we have rom-com, Action and sc-fi
  • In dramas we have korean and chinesi dramas
  • By clicking on the favourite movie we can see the movie in the corresponding links provided.
